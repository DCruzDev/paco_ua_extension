# paco_ua_extension
Extensão que adiciona funcionalidade ao PACO.

Ainda em fase inicial.

Features atuais:
* Mostra a média ponderada na página plano curricular

Features para adicionar:
* Adicionar exames ao google calendar com um clique
* ...
